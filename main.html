Question-3: What is event delegation in the DOM?

EvenDelegation is a technique in js where a parent element handles events for its child elements, even is the children are added dynamically after the page loads.

Question-4: What is event bubblink in the DOM?

Event Bubbling is a concept when we call a event and that event bubbles up to its parent and ancestor elements in the dom until it gets too the root element.
